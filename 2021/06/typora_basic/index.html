开始

  • ### 多级标题

几级标题：几个井号 + 空格（如：# 一级标题）或者 CTRL + 数字键

  • ### 引用

创建引用：> + 空格

退出引用：shift + tab

  • ### 无序列表

创建无序列表：ctrl + shift + ]

生成子列表：tab

回退：shift + tab

  • ### 有序列表

创建无序列表：ctrl + shift + [

生成子列表：tab

回退：shift + tab

  • ### 链接

[文字] +（网址）如百度

  • ### 图片

为了更好的管理，要将图片和文档放在一个文件夹下，所以要先设置一下typora

  1. 打开文件
  2. 打开偏好设置
  3. 选择图像
  5. 重启typora

之后在使用图片时，直接拖拽图片即可。而且图片会自动存放在与file.md同目录下的img文件夹

  • 划重点

    1. 高亮
      1. 设置：在偏好设置的markdown中打开高亮，重启使用。
      2. 使用：==文字==
    2. 加粗

    使用：** 文字 **（无空格）或者 ctrl + b

    1. ##### 斜体

    使用：* 文字 * （无空格）

    1. ##### 类高亮

    使用：· 文字 · （无空格）（符号为飘号，ESC键下面的）

  • 表格

使用：ctrl + t

  • ### 代码块

使用：三个波浪号 + 回车 （可在弹框中选择代码语言）

  int index = 0;
  • ### 删除线

使用：~~ 文字 ~~ （无空格）
